# 需求文档 — 超声教学资源管理系统

## 项目概述
本应用内置于彩超工作站中，用于浏览和播放超声诊断教学课件、视频、PDF 等教学资源。本质是将存放了大量教学资料的文件夹做成一个专业的软件应用，提供高效的浏览和播放体验。

## 核心需求

### 功能需求
1. **文件夹浏览**：支持浏览本地文件夹，显示目录树、文件网格，面包屑路径导航
2. **视频播放**：支持播放 .mp4/.avi/.mov/.mkv 等常见视频格式，带控制栏
3. **图片查看**：支持查看 .jpg/.png/.bmp 等常见图片格式，前后翻页
4. **PDF 查看**：支持查看 .pdf 课件，翻页、缩放
5. **PPT/Word**：调用系统 Office 打开 .ppt/.pptx/.doc/.docx 文件
6. **文件类型图标**：不同类型文件显示不同的视觉图标
7. **搜索过滤**：在当前文件夹内按文件名搜索
8. **收藏夹**：收藏常用文件夹/文件，快速访问
9. **最近打开**：自动记录最近打开的文件

### 非功能需求
1. **离线运行**：所有功能在无网络环境下正常工作
2. **便携部署**：打包为单个 .exe，无需安装，无需管理员权限
3. **大文件支持**：支持流畅播放 1GB+ 视频文件（流式传输）
4. **响应速度**：文件夹浏览响应 < 500ms
5. **兼容性**：Windows 10/11 系统
