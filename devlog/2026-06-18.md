## 解决的问题
1. **Electron 二进制下载失败**：GitHub 被墙 → 使用 npmmirror 镜像下载 → zip 已缓存但未自动解压 → 手动 unzip 解决
2. **require('electron') 返回字符串而非 API 对象**：系统环境变量 `ELECTRON_RUN_AS_NODE=1` 导致 Electron 以纯 Node.js 模式运行 → `unset ELECTRON_RUN_AS_NODE` 后正常
3. **npm start 脚本引号转义问题**：Windows 上 npm 运行 bash -c 时引号被破坏 → 改用 bash start.sh 脚本解决
